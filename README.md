
# Craighead Building Supplies - Netlify/React Bundle

This is a Vite + React project skeleton for Craighead.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Netlify

- Build command: `npm run build`
- Publish directory: `dist`

## Catalogue

Backend & catalogue seeds should be wired separately, using your SQL files and API implementation.
